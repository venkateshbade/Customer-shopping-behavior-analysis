# 📊 Customer Shopping Behavior Analysis

## 📌 Overview

This project analyzes customer transaction data to uncover patterns in purchasing behavior, product performance, and customer segments. The goal is to generate actionable insights that help businesses improve decision-making, increase revenue, and optimize marketing strategies.

---

## 🎯 Business Problem

Businesses collect large volumes of customer data but often fail to extract meaningful insights. Without proper analysis, companies struggle to:

* Identify high-value customers
* Optimize product offerings
* Improve customer retention
* Maximize revenue

This project addresses these challenges by analyzing customer shopping behavior using data analytics techniques.

---

## 🎯 Objectives

* Identify top-selling products and categories
* Analyze customer purchasing patterns
* Segment customers based on behavior
* Evaluate the impact of discounts on sales
* Compare spending patterns across customer groups
* Analyze revenue contribution by demographics

---

## 🛠️ Tech Stack

* **SQL** – Data querying and analysis
* **Python** – Data processing (Pandas, NumPy)
* **Visualization Tools** – Power BI / Tableau
* **Jupyter Notebook** – Analysis and reporting

---

## 📂 Dataset Description

The dataset contains customer transaction details, including:

* Customer ID
* Gender
* Age / Age Group
* Product Category
* Item Purchased
* Purchase Amount
* Review Rating
* Shipping Type
* Subscription Status
* Previous Purchases
* Discount Applied

---

## 🔍 Key Business Questions

The project answers the following key questions:

1. What is the total revenue generated by male vs female customers?
2. Which customers used discounts but still spent above average?
3. Which products have the highest average review ratings?
4. How does purchase amount vary by shipping type?
5. Do subscribed customers spend more than non-subscribers?
6. Which products are most frequently purchased with discounts?
7. How can customers be segmented based on purchasing behavior?
8. What are the top 3 products in each category?
9. Are repeat buyers more likely to subscribe?
10. What is the revenue contribution of each age group?

---

## 📊 Key Analysis Performed

* Data cleaning and preprocessing
* Exploratory Data Analysis (EDA)
* Customer segmentation (New, Returning, Loyal)
* Product performance analysis
* Discount impact analysis
* Revenue analysis by demographic segments
* Window functions for ranking products

## 📈 Insights (Example)
